# 서울 생활인구 데이터 지도 시각화 (Folium) 구현 완료

요청하신 대로 기존 대시보드에 **Folium 코로플리스(Choropleth) 맵**을 활용한 시간대별 생활인구 지도 시각화 기능을 성공적으로 추가했습니다.

## 구현된 주요 기능

1. **동적 GeoJSON 연동**
   - 워크스페이스 내에 GeoJSON 파일이 없는 제약을 해결하기 위해, 깃허브에 공개된 서울시 행정구역 GeoJSON(구 단위, 동 단위) 데이터를 동적으로 가져오도록 구현했습니다.

2. **Map (지도 시각화) 탭 추가**
   - 좌측 사이드바에 4번째 메뉴인 **Map (지도 시각화)**가 추가되었습니다.
   - **시간대 슬라이더**: 0시부터 23시까지 드래그하여 특정 시간대의 생활인구만 필터링하여 지도를 실시간으로 업데이트합니다.
   - **구별 / 동별 선택**: 인구를 '구' 단위로 뭉쳐서 볼지, '동' 단위로 쪼개서 볼지 선택할 수 있습니다.
   - **집계 방식 선택**: 해당 지역의 인구 '합계(Sum)'를 볼지, '평균(Mean)' 밀도를 볼지 선택 가능합니다.

3. **시각화 레이아웃**
   - 화면 **왼쪽**에는 빨간색-노란색(YlOrRd) 그라데이션으로 표현된 코로플리스 지도가 표시됩니다. 인구가 밀집된 지역일수록 붉은색으로 나타납니다.
   - 화면 **오른쪽**에는 지도에 표시된 데이터를 수치로 정확하게 확인할 수 있도록, 인구수가 높은 순서대로 정렬된 데이터 표(Table)를 함께 제공합니다.

## 🚀 확인 방법

현재 띄워두신 대시보드 브라우저(`http://192.168.219.100:8501`)에서 **새로고침(F5)** 버튼을 누르시면, 좌측 메뉴 맨 아래에 **"Map (지도 시각화)"** 탭이 나타납니다! 해당 탭을 클릭해서 지도를 조작해 보세요!
