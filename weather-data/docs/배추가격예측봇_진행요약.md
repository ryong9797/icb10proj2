# 스마트 장바구니 물가 예측 봇 — 진행 요약

> 작성일 2026-06-22 · 팀 공유용 (보안 정보 제외)

## 1. 프로젝트 개요

- **목표**: 농산물(배추) 가격을 예측해 소비자에게 "지금 살까 / 기다릴까"를 알려주는 카카오톡 챗봇 서비스.
- **핵심 가설**: 농산물 가격 급등은 당일 날씨가 아니라 **1~2개월 전 여름철 폭염·집중호우의 누적 효과(Time-Lag)** 가 시차를 두고 반영된 결과다.
- **타겟**: 1인 가구·일반 소비자. 복잡한 데이터 대신 직관적 구매 타이밍 알림.

## 2. 핵심 가설 검증 결과 ✅

데이터로 가설이 입증되었습니다.

- 시차 상관분석: 당일 기온-가격 상관 0.36 < **30일 전 0.43** (1개월 전 기온이 당일보다 가격과 더 강하게 연관).
- 모델 변수 중요도: 날씨만 사용 시 **45일 전 평균기온**이 1위.
- 결론: 여름 고온이 작황에 시차를 두고 반영되어 가을 가격을 끌어올리는 메커니즘 확인.

## 3. 데이터 파이프라인

| 데이터 | 출처 | 기간 | 비고 |
|--------|------|------|------|
| 날씨(기온·강수·일조) | 기상청 ASOS | 2021~현재 | 서울 등 5개 지점, 일별 |
| 배추 도매가 | KAMIS | 2021~현재 | 5개 지역, 평일 일별 |
| 배추 반입량(공급) | 가락시장 | 2021~현재 | 일별, 공급량 지표 |

- 전처리: 시차변수(30/45/60일) + 이동평균(7/14일) + 가격 자기상관 + 계절(월) 변수 생성.
- 결측치 처리: 가격은 직전 평일값(ffill), 강수량 결측은 0.
- 데이터 누수 방지: 시계열 순서 유지(셔플 금지), 이동평균은 당일 제외.

## 4. 예측 모델

XGBoost 기반, 두 가지 예측 호라이즌 + 조기경보 트랙.

| 모델 | 용도 | 성능(MAPE) |
|------|------|-----------|
| 단기(7일 후) | 주간 구매 타이밍 | 약 20% |
| 중기(30일 후) | 한 달 뒤 폭등 경보 | 약 27% |
| 조기경보(분류) | 급등 사전 알림 | 정밀도 78% / 재현율 73% |

- 평가지표: RMSE·MAPE·WAPE (현업 친화 지표 포함).
- 매일 최신 데이터로 자동 재학습되는 파이프라인 구축.

## 5. 주요 분석 인사이트

1. **시차(Time-Lag) 효과 입증**: 가격 예측의 핵심 변수는 1~2개월 전 날씨.
2. **외부 데이터는 용도에 맞게**: 가락시장 반입량(공급량)은 조기경보엔 효과 없었으나, **정밀 가격예측에선 MAPE를 1~3%p 개선**. "외부 데이터 무조건 추가"가 아니라 검증 기반으로 선택.
3. **데이터 기간의 중요성**: 학습 데이터를 최신까지 확장하자 조기경보 재현율이 크게 향상(분포 안정화 효과).

## 6. 서비스 아키텍처

```
[데이터 수집] 기상청 / KAMIS / 가락시장
      ↓
[전처리 + 시차변수 생성]
      ↓
[XGBoost 모델: 단기·중기·조기경보]
      ↓
[API 서버 (FastAPI)]  ← 클라우드 배포
      ↓
[카카오톡 챗봇]  → 카드형 응답(예측 + 대체재 추천)
```

- 챗봇 응답: 가격 예측 카드 + (가격 상승 시) 대체재(양배추·얼갈이배추) 추천 카드.
- 기술 스택: Python, pandas, XGBoost, FastAPI, 클라우드 PaaS, 카카오 i 오픈빌더.
- 모든 인증키는 **환경변수로 분리 관리**(코드/저장소 비노출), 저장소는 비공개.
